Transport Consolidated

Transport Consolidated is a Full stack web application using the MERN stack. It was designed as a solution for businesses finding trucks to move there loads. A modern front end design with a backend built on newer technoliges, it has the potential to grow to a production ready webservice. This is an ever growing project and will continue to get updates as time moves along.
