Transport Consolidated

Transport Consolidated is a Full stack web application using the MERN stack. It was designed as a solution for businesses finding trucks to move there products. Created to be user friendly with a simple yet elegant design, while offereing a nonSQL database solution. It is currently in a testing stage, so feel free to use dummy information to give it a try. I am excited about the progress of this project and look foward to adding updates.
